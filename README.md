# AllWeatherOS

**경제의 모든 계절에 대응하는 공익 금융 지식 블로그**

AllWeatherOS는 한국 개인 투자자가 ISA·연금저축·IRP 같은 절세계좌와 일반계좌를 함께 이해하고, ETF·채권·금·원자재·리츠·현금성 자산·글로벌 주식 등 다양한 금융상품을 활용해 장기 자산배분을 설계하도록 돕는 GitHub Pages 기반 공익 금융 리포지토리입니다.

## Editorial Promise

- 특정 종목 매수·매도 추천이 아닌 **교육 목적의 금융 정보**를 제공합니다.
- 뉴스룸 편집국장 관점으로 **사실, 맥락, 한계, 출처**를 분리합니다.
- AEO·GEO 친화 구조로 **AI가 인용하기 쉬운 정의·요약·FAQ·표·출처**를 포함합니다.
- 모든 시황성 글은 날짜와 기준시각을 명시합니다.

## Current Focus

1. 절세계좌 3총사: ISA, 연금저축, IRP
2. 국내상장 해외 ETF와 글로벌 자산배분
3. 금리·환율·원자재·채권·리츠·현금성 자산
4. 올웨더·리스크 패리티·잃지 않는 포트폴리오 철학

## Repository Structure

```text
_posts/        발행 글
_data/         관심 ETF, 출처, 편집 기준 데이터
categories/    카테고리 인덱스
templates/     기사·리포트 템플릿
scripts/       향후 데이터 수집/자동화 스크립트
```

## Disclaimer

본 리포지토리의 콘텐츠는 경제·금융 교육 목적의 정보이며 투자 권유가 아닙니다. 금융상품 투자에는 원금 손실 가능성이 있습니다.
