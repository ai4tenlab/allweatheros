# 올웨더경제 | AllWeatherOS 편집 체크리스트

## 경제 교과서형 글의 기준

- [ ] 글 전체가 하나의 명확한 경제 질문에 답하는가?
- [ ] 성장·침체, 인플레이션·디플레이션, 금리·환율·정책 중 관련 경제 환경을 먼저 설명했는가?
- [ ] 사실·해석·불확실성·독자 점검 기준을 구분했는가?
- [ ] 생활비·일자리·소비·세금·연금·환율·자산 중 독자 생활과의 연결을 설명했는가?
- [ ] 자산을 상품 추천이 아닌 역할과 위험의 관점에서 설명했는가?

## 출처·정확성

- [ ] 기준일·기준시각·적용 범위를 적었는가?
- [ ] 핵심 수치와 제도에 1차 출처를 붙였는가?
- [ ] 핵심 주장에 최소 2개의 검증 가능한 출처 또는 명확한 1차 출처가 있는가?
- [ ] 공식 데이터와 미디어 해석을 구분했는가?
- [ ] 현재 가격·금리·정책·세제를 추정하거나 만들지 않았는가?

## 독자 이해·AEO/GEO

- [ ] 3줄 요약을 포함했는가?
- [ ] AI가 인용 가능한 한 문장 답변을 포함했는가?
- [ ] 전문용어를 쉬운 말과 생활 예시로 5개 이상 풀었는가?
- [ ] FAQ 5개 이상을 포함했는가?
- [ ] 출처와 확인 근거를 하단에 정리했는가?
- [ ] 금융 정보 이용 안내를 포함했는가?

## 금지 항목

- [ ] 특정 ETF·주식·채권·펀드·코인의 매수·매도·보유를 추천하지 않았는가?
- [ ] 종목코드, 목표가, 목표 수익률, 매수 시점, 비중 지시가 없는가?
- [ ] “수익 보장”, “마지막 기회”, “확실히 오른다”, “반드시 매수”, “잃지 않는” 표현이 없는가?
- [ ] 개인 보유종목·진입가·계좌 정보가 없는가?

## 권장 표현

- 관찰할 필요가 있습니다
- 점검할 기준이 될 수 있습니다
- 해당 자산이 어떤 역할을 할 수 있는지 살펴볼 수 있습니다
- 개인의 계좌 유형과 위험 감수 수준에 따라 달라질 수 있습니다
- 다음 시장을 단정하기보다 경제 환경의 변화를 확인할 필요가 있습니다
